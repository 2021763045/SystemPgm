## fork() 정리

### 1. 개요

`fork()`는 Linux/Unix 계열 운영체제에서 **새로운 프로세스(자식 프로세스)를 생성하는 시스템 호출**이다. 기존 프로세스(부모 프로세스)를 복제하여 자식 프로세스를 만들며, 멀티프로세스 프로그램을 구현할 때 사용된다.

---

### 2. fork() 함수

헤더 파일

```c
#include <unistd.h>
```

함수 원형

```c
pid_t fork(void);
```

---

### 3. fork()의 반환값

`fork()` 호출 후 부모 프로세스와 자식 프로세스는 서로 다른 반환값을 받는다.

| 반환값     | 의미                  |
| ------- | ------------------- |
| 0       | 자식 프로세스             |
| 양수(PID) | 부모 프로세스 (자식 PID 반환) |
| -1      | 프로세스 생성 실패          |

---

### 4. 동작 과정

1. 부모 프로세스가 `fork()` 호출
2. 운영체제가 부모 프로세스를 복제
3. 자식 프로세스 생성
4. 부모와 자식 프로세스가 각각 실행

구조

```text
부모 프로세스
      │
    fork()
      │
 ┌────┴────┐
 │         │
부모      자식
```

---

### 5. 예제 코드

```c
#include <stdio.h>
#include <unistd.h>

int main() {
    pid_t pid;

    pid = fork();

    if (pid == 0)
        printf("자식 프로세스\n");
    else
        printf("부모 프로세스\n");

    return 0;
}
```

실행 결과 예시

```text
부모 프로세스
자식 프로세스
```

※ 실행 순서는 운영체제 스케줄링에 따라 달라질 수 있다.

---

### 6. fork()의 특징

* 부모 프로세스를 복제하여 자식 프로세스 생성
* 부모와 자식은 독립적인 프로세스
* 각각 고유한 PID를 가짐
* 코드와 데이터는 복사되지만 별도의 메모리 공간 사용
* 생성 이후 서로 독립적으로 실행

---

### 7. 관련 함수

| 함수       | 설명            |
| -------- | ------------- |
| `fork()` | 자식 프로세스 생성    |
| `wait()` | 자식 프로세스 종료 대기 |
| `exec()` | 새로운 프로그램 실행   |
| `exit()` | 프로세스 종료       |

---

### 정리

| 항목     | 내용                 |
| ------ | ------------------ |
| 목적     | 자식 프로세스 생성         |
| 반환값 0  | 자식 프로세스            |
| 반환값 양수 | 부모 프로세스            |
| 반환값 -1 | 생성 실패              |
| 특징     | 부모 프로세스 복제 후 독립 실행 |
