## Linux 기본 명령어 및 경로

### 1. Linux 기본 명령어

#### 파일 및 디렉터리 관리

| 명령어     | 설명                  |
| ------- | ------------------- |
| `pwd`   | 현재 작업 중인 디렉터리 경로 출력 |
| `ls`    | 파일 및 디렉터리 목록 확인     |
| `cd`    | 디렉터리 이동             |
| `mkdir` | 새 디렉터리 생성           |
| `rmdir` | 빈 디렉터리 삭제           |
| `touch` | 빈 파일 생성             |
| `cp`    | 파일 또는 디렉터리 복사       |
| `mv`    | 파일 이동 또는 이름 변경      |
| `rm`    | 파일 삭제               |
| `rm -r` | 디렉터리와 내부 파일 삭제      |

#### 파일 내용 확인

| 명령어    | 설명                |
| ------ | ----------------- |
| `cat`  | 파일 전체 내용 출력       |
| `more` | 파일 내용을 페이지 단위로 확인 |
| `less` | 파일 내용을 스크롤하며 확인   |
| `head` | 파일 앞부분 출력         |
| `tail` | 파일 뒷부분 출력         |

#### 시스템 관련

| 명령어       | 설명            |
| --------- | ------------- |
| `whoami`  | 현재 로그인 사용자 확인 |
| `date`    | 현재 날짜 및 시간 확인 |
| `clear`   | 터미널 화면 정리     |
| `history` | 사용한 명령어 기록 확인 |
| `exit`    | 터미널 종료        |

#### 권한 관련

| 명령어     | 설명          |
| ------- | ----------- |
| `chmod` | 파일 권한 변경    |
| `chown` | 파일 소유자 변경   |
| `sudo`  | 관리자 권한으로 실행 |

---

### 2. Linux 경로(Path)

| 기호   | 의미         |
| ---- | ---------- |
| `/`  | 루트 디렉터리    |
| `.`  | 현재 디렉터리    |
| `..` | 상위 디렉터리    |
| `~`  | 사용자 홈 디렉터리 |

예시

```bash
cd ~
```

→ 홈 디렉터리로 이동

```bash
cd ..
```

→ 상위 디렉터리로 이동

```bash
cd .
```

→ 현재 디렉터리 유지
