## 프로세스(Process)

### 1. 개요

프로세스(Process)는 실행 중인 프로그램을 의미한다. 단순히 저장되어 있는 프로그램이 아니라, 메모리에 적재되어 CPU의 자원을 할당받아 실행되고 있는 상태를 말한다.

예를 들어, 메모장 프로그램을 실행하면 메모장 프로그램이 하나의 프로세스가 된다.

---

### 2. 프로그램과 프로세스의 차이

| 구분 | 프로그램       | 프로세스        |
| -- | ---------- | ----------- |
| 의미 | 저장된 실행 파일  | 실행 중인 프로그램  |
| 위치 | 디스크(저장장치)  | 메모리         |
| 상태 | 정적(Static) | 동적(Dynamic) |

---

### 3. 프로세스의 구성 요소

프로세스는 메모리 내에서 다음과 같은 영역으로 구성된다.

| 영역         | 설명              |
| ---------- | --------------- |
| Code(Text) | 실행할 프로그램 코드     |
| Data       | 전역 변수, 정적 변수    |
| Heap       | 동적 메모리 영역       |
| Stack      | 함수 호출, 지역 변수 저장 |

---

### 4. 프로세스 상태

프로세스는 실행 과정에서 여러 상태를 가진다.

```text
생성(New)
   ↓
준비(Ready)
   ↓
실행(Running)
   ↓
대기(Waiting)
   ↓
종료(Terminated)
```

#### 상태 설명

| 상태         | 설명           |
| ---------- | ------------ |
| New        | 프로세스 생성      |
| Ready      | CPU 할당 대기    |
| Running    | CPU에서 실행 중   |
| Waiting    | 입출력 등 이벤트 대기 |
| Terminated | 실행 종료        |

---

### 5. 프로세스 관련 시스템 호출

| 함수       | 설명            |
| -------- | ------------- |
| `fork()` | 자식 프로세스 생성    |
| `exec()` | 새로운 프로그램 실행   |
| `wait()` | 자식 프로세스 종료 대기 |
| `exit()` | 프로세스 종료       |

예시

```c
pid_t pid = fork();
```

* 부모 프로세스를 복제하여 자식 프로세스 생성
* 부모와 자식은 각각 독립적으로 실행

---

### 6. Linux 프로세스 관리 명령어

| 명령어      | 설명             |
| -------- | -------------- |
| `ps`     | 실행 중인 프로세스 확인  |
| `top`    | 실시간 프로세스 상태 확인 |
| `pstree` | 프로세스 계층 구조 확인  |
| `kill`   | 프로세스 종료        |
| `jobs`   | 백그라운드 작업 확인    |

예시

```bash
ps -ef
```

→ 현재 실행 중인 모든 프로세스 확인

```bash
kill 1234
```

→ PID가 1234인 프로세스 종료

---

### 7. PID(Process ID)

PID는 운영체제가 각 프로세스에 부여하는 고유 번호이다.

특징

* 프로세스마다 서로 다른 PID를 가짐
* 프로세스 관리 및 제어에 사용
* `ps` 명령어로 확인 가능

---

### 정리

| 항목    | 내용                                  |
| ----- | ----------------------------------- |
| 프로세스  | 실행 중인 프로그램                          |
| PID   | 프로세스 고유 번호                          |
| 주요 상태 | Ready, Running, Waiting, Terminated |
| 생성    | `fork()`                            |
| 실행    | `exec()`                            |
| 종료    | `exit()`, `kill`                    |
| 확인    | `ps`, `top`                         |
