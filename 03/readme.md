## Linux 파일 관련 명령어 및 도움말 명령어

### 1. 파일 관련 명령어

#### 파일 생성 및 삭제

| 명령어           | 설명             |
| ------------- | -------------- |
| `touch 파일명`   | 빈 파일 생성        |
| `rm 파일명`      | 파일 삭제          |
| `rm -r 디렉터리명` | 디렉터리와 내부 파일 삭제 |
| `mkdir 디렉터리명` | 디렉터리 생성        |
| `rmdir 디렉터리명` | 빈 디렉터리 삭제      |

#### 파일 복사 및 이동

| 명령어           | 설명       |
| ------------- | -------- |
| `cp 원본 대상`    | 파일 복사    |
| `cp -r 원본 대상` | 디렉터리 복사  |
| `mv 원본 대상`    | 파일 이동    |
| `mv 기존이름 새이름` | 파일 이름 변경 |

#### 파일 내용 확인

| 명령어           | 설명                |
| ------------- | ----------------- |
| `cat 파일명`     | 파일 전체 내용 출력       |
| `more 파일명`    | 페이지 단위로 내용 확인     |
| `less 파일명`    | 스크롤하며 내용 확인       |
| `head 파일명`    | 파일 앞부분 출력         |
| `tail 파일명`    | 파일 뒷부분 출력         |
| `tail -f 파일명` | 실시간으로 파일 변경 내용 확인 |

#### 파일 검색 및 정보 확인

| 명령어        | 설명             |
| ---------- | -------------- |
| `find`     | 파일 및 디렉터리 검색   |
| `locate`   | 파일 위치 검색       |
| `file 파일명` | 파일 종류 확인       |
| `wc 파일명`   | 행, 단어, 문자 수 확인 |
| `stat 파일명` | 파일 상세 정보 확인    |

#### 권한 관련

| 명령어     | 설명               |
| ------- | ---------------- |
| `chmod` | 파일 권한 변경         |
| `chown` | 파일 소유자 변경        |
| `ls -l` | 파일 권한 및 상세 정보 확인 |

### 2. 도움말 관련 명령어

Linux는 대부분의 명령어에 대한 도움말을 제공한다.

#### man (Manual)

명령어의 공식 매뉴얼을 확인한다.

예시

```bash
man ls
```

특징

* 가장 자세한 도움말 제공
* 사용법, 옵션, 설명 확인 가능

---

#### help

쉘(Bash) 내장 명령어의 도움말을 확인한다.

예시

```bash
help cd
```

특징

* Bash 내장 명령어 설명 제공
* 간단하고 빠르게 확인 가능

---

#### --help 옵션

명령어 사용법을 간단하게 확인한다.

예시

```bash
ls --help
```

특징

* 자주 사용하는 옵션 확인 가능
* 빠른 사용법 참고에 유용

---

#### whatis

명령어의 간단한 설명을 확인한다.

예시

```bash
whatis cp
```

출력 예시

```bash
cp (1) - copy files and directories
```

---

#### apropos

키워드를 이용해 관련 명령어를 검색한다.

예시

```bash
apropos copy
```

특징

* 명령어 이름이 기억나지 않을 때 유용
* 관련 명령어 목록 제공

---

### 정리

| 명령어       | 용도            |
| --------- | ------------- |
| `touch`   | 파일 생성         |
| `cp`      | 파일 복사         |
| `mv`      | 파일 이동 및 이름 변경 |
| `rm`      | 파일 삭제         |
| `cat`     | 파일 내용 확인      |
| `find`    | 파일 검색         |
| `chmod`   | 파일 권한 변경      |
| `man`     | 상세 매뉴얼 확인     |
| `help`    | 쉘 내장 명령어 도움말  |
| `--help`  | 간단한 사용법 확인    |
| `whatis`  | 명령어 한 줄 설명    |
| `apropos` | 관련 명령어 검색     |
