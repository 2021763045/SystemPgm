## 입출력 버퍼(Input/Output Buffer)

### 1. 개요

입출력 버퍼(Buffer)는 데이터 입출력 시 데이터를 임시로 저장하는 메모리 공간이다. 프로그램과 입출력 장치(키보드, 모니터, 파일 등) 사이에서 데이터 전송 효율을 높이기 위해 사용된다.

---

### 2. 버퍼를 사용하는 이유

* 입출력 장치의 속도 차이 보완
* 시스템 호출 횟수 감소
* 프로그램 성능 향상
* 데이터 처리 효율 증가

예를 들어 파일에 데이터를 한 글자씩 저장하는 대신, 버퍼에 모아두었다가 한 번에 저장하면 속도가 더 빨라진다.

---

### 3. 버퍼링(Buffering) 종류

#### ① 완전 버퍼링 (Fully Buffered)

버퍼가 가득 찼을 때 한 번에 입출력을 수행한다.

특징

* 파일 입출력에서 주로 사용
* 입출력 효율이 가장 높음
* 버퍼가 가득 차기 전까지 출력되지 않을 수 있음

---

#### ② 라인 버퍼링 (Line Buffered)

개행 문자(`\n`)를 만날 때마다 버퍼 내용을 출력한다.

특징

* 터미널(콘솔) 출력에서 주로 사용
* 한 줄 단위로 출력 수행

예시

```c
printf("Hello\n");
```

→ `\n`이 포함되어 즉시 출력된다.

---

#### ③ 비버퍼링 (Unbuffered)

버퍼를 사용하지 않고 즉시 입출력을 수행한다.

특징

* 가장 빠르게 출력 결과 확인 가능
* 입출력 효율은 낮음

예시

* 표준 오류 스트림(`stderr`)

---

### 4. 버퍼 비우기

#### fflush()

버퍼에 저장된 데이터를 즉시 출력하거나 파일에 기록한다.

```c
fflush(stdout);
```

특징

* 출력 버퍼를 강제로 비움
* 즉시 출력이 필요한 경우 사용

---

### 5. 표준 입출력 스트림

| 스트림      | 설명          |
| -------- | ----------- |
| `stdin`  | 표준 입력 (키보드) |
| `stdout` | 표준 출력 (화면)  |
| `stderr` | 표준 에러 출력    |

---

### 6. 버퍼 사용 예시

```c
#include <stdio.h>

int main() {
    printf("Hello");
    fflush(stdout);

    return 0;
}
```

위 코드에서 `fflush(stdout)`을 사용하면 버퍼에 저장된 내용이 즉시 화면에 출력된다.

---

### 정리

| 종류         | 특징           |
| ---------- | ------------ |
| 완전 버퍼링     | 버퍼가 가득 차면 출력 |
| 라인 버퍼링     | 줄바꿈 시 출력     |
| 비버퍼링       | 즉시 출력        |
| `fflush()` | 버퍼 강제 출력     |
