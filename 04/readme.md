## C 언어 파일 입출력(File I/O) 정리

### 1. 개요

파일 입출력(File I/O)은 프로그램에서 데이터를 파일에 저장하거나 파일로부터 읽어오는 기능이다. C 언어에서는 `stdio.h` 라이브러리의 함수들을 사용하여 파일을 처리한다.

---

### 2. 파일 처리 과정

1. 파일 열기 (`fopen`)
2. 파일 읽기 또는 쓰기
3. 파일 닫기 (`fclose`)

기본 구조

```c
FILE *fp;

fp = fopen("data.txt", "r");

/* 파일 작업 */

fclose(fp);
```

---

### 3. 파일 열기 함수

#### fopen()

파일을 열고 파일 포인터를 반환한다.

```c
FILE *fp;
fp = fopen("파일명", "모드");
```

#### 파일 열기 모드

| 모드     | 설명               |
| ------ | ---------------- |
| `"r"`  | 읽기 전용            |
| `"w"`  | 쓰기 전용 (기존 내용 삭제) |
| `"a"`  | 이어쓰기             |
| `"r+"` | 읽기/쓰기            |
| `"w+"` | 읽기/쓰기 (기존 내용 삭제) |
| `"a+"` | 읽기/이어쓰기          |

---

### 4. 파일 쓰기 함수

#### fprintf()

파일에 형식화된 데이터를 저장한다.

```c
fprintf(fp, "%s %d", name, age);
```

#### fputc()

문자 1개를 파일에 저장한다.

```c
fputc('A', fp);
```

#### fputs()

문자열을 파일에 저장한다.

```c
fputs("Hello C", fp);
```

---

### 5. 파일 읽기 함수

#### fscanf()

파일에서 형식화된 데이터를 읽는다.

```c
fscanf(fp, "%s %d", name, &age);
```

#### fgetc()

파일에서 문자 1개를 읽는다.

```c
char ch;
ch = fgetc(fp);
```

#### fgets()

한 줄의 문자열을 읽는다.

```c
char str[100];
fgets(str, sizeof(str), fp);
```

---

### 6. 파일 닫기 함수

#### fclose()

열려 있는 파일을 닫는다.

```c
fclose(fp);
```

파일 사용 후 반드시 닫아야 하며, 메모리 누수를 방지할 수 있다.

---

### 정리

| 함수          | 기능          |
| ----------- | ----------- |
| `fopen()`   | 파일 열기       |
| `fclose()`  | 파일 닫기       |
| `fprintf()` | 파일에 데이터 쓰기  |
| `fscanf()`  | 파일에서 데이터 읽기 |
| `fputc()`   | 문자 1개 쓰기    |
| `fgetc()`   | 문자 1개 읽기    |
| `fputs()`   | 문자열 쓰기      |
| `fgets()`   | 문자열 읽기      |
